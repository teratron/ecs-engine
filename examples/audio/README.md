# Example: Audio System

Examples in this category focus on playing background music and 3D spatial audio emitters.

## Planned Examples

- **`play_sound`**: Basic trigger-based sound effect playback.
- **`spatial_audio`**: 3D audio positioning that changes based on entity transforms and listener location.
- **`audio_control`**: Controlling volume, panning, and playback speed at runtime.
- **`music_loop`**: Managing seamless background music loops and transitions.

## Running the Example

```bash
go run ./examples/audio
```

## Testing

