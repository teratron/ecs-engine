# Example: Diagnostics & Debugging

Examples in this category focus on performance monitoring, visual debugging, and runtime profiling.

## Planned Examples

- **`fps_overlay`**: Displaying a real-time FPS counter and frame time diagnostics.
- **`gizmos`**: Drawing debug shapes (lines, circles, boxes) in world space for visualization.
- **`profiling`**: Integrating with Go's `pprof` and internal system timers to identify bottlenecks.
- **`log_view`**: Capturing and displaying engine logs in an on-screen debug console.
- **`entity_inspector`**: A basic runtime inspector to view and modify entity components.

## Running the Example

```bash
go run ./examples/diagnostic
```

## Testing

```bash
go test ./examples/diagnostic/...
```
