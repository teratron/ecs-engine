# Example: Physics System

Examples in this category focus on rigid body dynamics, collision detection, and physics-based movement.

## Planned Examples

- **`gravity`**: Simple falling objects using the rigid body system.
- **`collision`**: Detecting and responding to collisions between different shapes.
- **`rigid_body`**: Applying forces, impulses, and torques to simulate realistic movement.
- **`joints`**: Connecting entities with physical constraints (hinges, sliders, springs).
- **`raycasting`**: Performing physics queries like raycasts and overlap tests.

## Running the Example

```bash
go run ./examples/physics
```

## Testing

```bash
go test ./examples/physics/...
```
