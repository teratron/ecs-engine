# Example: Networking & Replication

Examples in this category focus on multiplayer synchronization, entity replication, and network transport.

## Planned Examples

- **`client_server`**: Setting up a basic client-server connection and messaging.
- **`replication`**: Automatically synchronizing entity components across the network.
- **`rpc`**: Using Remote Procedure Calls to trigger logic on other peers.
- **`prediction`**: Implementing client-side prediction and server reconciliation for smooth movement.
- **`snapshot_interpolation`**: Smoothly interpolating entity states between received network snapshots.

## Running the Example

```bash
go run ./examples/networking
```
