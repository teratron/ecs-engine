# Example: 3D Rendering

Examples in this category focus on the 3D graphics pipeline, including meshes, lighting, and materials.

## Planned Examples

- **`mesh`**: Loading and rendering static 3D meshes (OBJ/GLTF).
- **`lighting`**: Basic point, directional, and ambient lighting models.
- **`pbr_material`**: Demonstrating Physically Based Rendering (PBR) materials with metalness and roughness.
- **`perspective_camera`**: Setting up a 3D perspective camera with frustum culling.
- **`skybox`**: Implementing a cube-mapped skybox environment.

## Running the Example

```bash
go run ./examples/3d
```

## Testing

```bash
go test ./examples/3d/...
```
