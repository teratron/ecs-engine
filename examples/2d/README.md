# Example: 2D Rendering

Examples in this category focus on the 2D graphics pipeline, including sprites, orthographic cameras, and 2D transforms.

## Planned Examples

- **`sprite`**: Rendering basic 2D sprites with positions and rotations.
- **`textured_quad`**: Demonstrating texture mapping and UV coordinates on 2D planes.
- **`orthographic_camera`**: Setting up a camera for 2D viewports with zooming and panning.
- **`parallax`**: Implementing multi-layered scrolling backgrounds using different camera offsets.
- **`shapes`**: Drawing primitive 2D shapes (rectangles, circles) using the vector graphics pipeline.

## Running the Example

```bash
go run ./examples/2d
```
