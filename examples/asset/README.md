# Example: Asset Management

Examples in this category focus on asynchronous asset loading, handle management, and hot reloading.

## Planned Examples

- **`asset_loading`**: Loading textures, sounds, and models asynchronously using handles.
- **`hot_reloading`**: Demonstrating live updates to assets when source files are modified.
- **`custom_loader`**: Implementing a custom asset loader for user-defined formats.
- **`scenes`**: Serializing and deserializing entity snapshots into scenes.
- **`asset_server`**: Advanced asset server configuration and folder watching.

## Running the Example

```bash
go run ./examples/asset
```
