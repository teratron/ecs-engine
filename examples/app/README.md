# Example: Application Framework

Examples in this category demonstrate the application lifecycle, plugin system, system scheduling, and state management.

## Planned Examples

- **`hello_world`**: The most basic engine application setup with a single system.
- **`plugins`**: Modularizing application logic into reusable plugins.
- **`app_states`**: Managing transitions between different game states (e.g., Menu, Playing, Paused).
- **`custom_runner`**: Implementing a custom game loop for specialized environments (e.g., headless servers).
- **`sub_apps`**: Running isolated sub-applications for parallel tasks like rendering or physics.

## Running the Example

```bash
go run ./examples/app
```

## Testing

```bash
go test ./examples/app/...
```
