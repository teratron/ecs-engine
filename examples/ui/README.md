# Example: UI System

Examples in this category focus on building user interfaces using the engine's built-in UI nodes and layout system.

## Planned Examples

- **`buttons`**: Creating interactive buttons with hover, press, and release states.
- **`layout`**: Demonstrating Flexbox-like layout behaviors for UI containers.
- **`text`**: Rendering dynamic text with different fonts and alignments.
- **`hud`**: Building a typical game Heads-Up Display (health bars, inventory).
- **`styles`**: Applying consistent themes and design tokens to UI elements.

## Running the Example

```bash
go run ./examples/ui
```

## Testing

```bash
go test ./examples/ui/...
```
